# `scripts/`

This folder contains deterministic starter-pack utility scripts only.

Included:

- `generate_synthetic.py`: regenerates `data/linear_gaussian.npz` and `data/moons.npz`
- `make_digits_split.py`: regenerates `data/digits_split_indices.npz`

These scripts are for reproducible data preparation only. They are not part of the model implementation.
