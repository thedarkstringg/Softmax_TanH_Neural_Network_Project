# `data/`

This folder contains only data assets.

Expected contents:

- `digits_data.npz`: digits features and labels stored in NumPy format
- `digits_split_indices.npz`: fixed train/validation/test indices
- `linear_gaussian.npz`: generated linear synthetic dataset
- `moons.npz`: generated nonlinear synthetic dataset

Related deterministic scripts live in `../scripts/`.

Students should treat these files as fixed inputs to the capstone, not as a prompt to search for alternative datasets.
